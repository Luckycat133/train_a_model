## 📊 数据概览

灵猫墨韵项目使用中国古典文献作为训练数据，主要包括：

- **古文经典**：论语、道德经、庄子等先秦诸子著作
- **诗词歌赋**：唐诗、宋词等经典诗词作品
- **古典术语**：专业术语词典用于专业领域生成
