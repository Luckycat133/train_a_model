# Model weights

Model binaries are intentionally not stored in Git.

A future external release must record:

- model configuration and parameter count;
- dataset name, version, and license;
- training seed and command;
- validation metrics and limitations;
- file size and SHA-256 checksum;
- durable external download URL.

Current status: no externally verified weight release is published. Historical tracked weights remain in old commits until a separately approved, backed-up history rewrite is performed.
